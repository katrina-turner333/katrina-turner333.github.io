Difference between HTML and CSS

In simple, non technical terms, you can think of HTML as the structure for a house. CSS on the other hand is the interior/exterior designer who comes to make your house look pretty. HTML is the builder, CSS is the artist. In the world of web design, HTML is responsible for dictating the content and structure of a webpage. CSS is responsible for modifying the design and display of HTML elements. Together, they produce a complete webpage interface.

Control Flow

In computer programming there is something called Control Flow.
Control Flow is a way to tell the execution how the program should run and jump from one section to another. CF refers to the order of execution of the flow.
For example, if you’re running a JavaScript code on a browser, the compiler will reach a point in accord that says if this is true, do something else. Or at some point it can be told to do something x number of times, before you move on to the next order of execution. It just refers to the order of execution of the code. How the program jumps form one section to another.

Control flow can be thought of as doing things in the best possible order in order to get the desired result.
For example, in life, the natural flow is to usually first go to school in order to go to university or to get a job. Our society follows this flow in order to achieve the desired outcome of either university or job. It would be too difficult for us to go to univeristy if we never went to school etc. Thats an example of control flow. We control the way we achieve things in life.

Loops are an example of Control Flow. See details below.

DOM

DOM is an acronym that stands for Document Object Model, AKA ‘application programming interface’. What this actually means is we can use the DOM to interact with webpages.

We can do things like add or remove content from an HTML document, can change content etc. This and more can be done through the DOM, very easily while your web page is being loaded in a web browser.

Here is an example of how you would interact with it:
1. Go to any website using Google Chrome, right click to button Inspect, open up the developer tools to access the DOM.
2. Next, in the contents area of the DOM, change the content text. Now you can straight away see the changes on the website.
However, this is not going to be permanent, but it is a cool trick to show your friends.

Difference between accessing data from arrays and objects

What makes an array are these square brackets: [ ] . They are just a set of data.

For example,
var arr= [1,true,0]
Inside an arrays you can put anything, such as a string, a number,, bullion, or another array.

Objects however use curly brackets: {} and have keys and properties.

For example:
Var obj= {
“name”: “Nute”,
“drink”: “Coke”
};

The above strings (another name for words in JavaScript code) “name” and “drink” are referred to as key values. They are on the left side. The above strings “Nute” and “Coke” are referred to as property values. They are on the right side. This is referred to as Javascript Object Notation (JSON).

Accessing Data


Arrays use a number, an index value to access an array item.

For example;
var students = [“Anne”, “Ben”, “Helena”];
alert( students[0] );

The above code will allow you to access the 0, which in an array stands for the first property. This one will access “Anne”. Objects use a key to access their properties.

For example;
var person = {
name: “Ben”,
age: 23
};
alert( person[‘name’]);

This will show you the name “Ben”.

Functions and why they are useful

In JavaScript functions are one of the fundamental building blocks. It is a procedure, a set of statements that performs a task or calculates a value. Functions are a special type of object. To use a function, you must define it somewhere in the scope (section of coding page that has the applicable code) from which you wish to call it (use it).

Here is an example of a simple function called square:
function square(number) {
return number * number;
}

The function square takes one parameter, called number. The function consists of one statement that says to return the parameter of the function (that is, number) multiplied by itself.
